# SpringBoot Testing
### Custom built API using Java 18 and Maven


## Run Locally

Clone the project

```bash
  git clone https://github.com/samiylo/springboot-testing.git
```

Go to the project directory

```bash
  cd demo
```

Install dependencies

```bash
  mvn clean install
```

Start the server

```bash
  mvn spring-boot:run
```
